# AxieScout
AxieScout is a personal project for monitoring Axie Infinity prices, traits, and market stats using the Ronin API. This tool is intended for personal research and portfolio tracking.

# Features
1. View current Axie market prices
2. Track Axie traits and rarity
3. Personal portfolio monitoring (no user data collected)

# Getting Started
This project is for personal use and can be run locally. Clone the repository to your machine:

Copy code
```bash git clone https://github.com/Kingcthulhu/AxieScout.git cd AxieScout python main.py```
